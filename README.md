# Lusi AI Assistant

A simple Flutter application that implements a chat interface for an AI assistant named Lusi.

## Features
- Clean, modern chat UI with Material 3 design.
- Differentiates between user and AI messages.
- Displays a typing indicator when the AI is "thinking".
- Voice support: Dictate messages to the AI assistant instead of typing.
- Responsive layout supporting multiple devices.

## Tech Stack
- Framework: Flutter
- Language: Dart
- UI: Material Design 3

## Setup and Run
1. Ensure you have Flutter installed (`flutter doctor`).
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your preferred device.
